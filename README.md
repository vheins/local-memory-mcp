# MCP Local Memory Service

An MCP (Model Context Protocol) service that provides long-term memory for AI coding agents.

## Documentation

- [Product Requirements Document (PRD)](docs/PRD.md)
- [Auto-Memory Heuristics & Scoping](docs/SPEC-heuristics.md)

## Overview

This project implements a local memory service using Node.js and SQLite with vector similarity search capabilities. It aims to solve the context loss problem in coding assistants by storing code facts, decisions, mistakes, and patterns.

For detailed requirements and specifications, please refer to the [PRD](docs/PRD.md).
